# Assignment version for Senior Engineers
This is a test assignment for the role of a Senior Platform Engineer (Infrastructure)

## Containerize an application
This repository contains a simple Flask application written in Python. Your goal is to improve this application to be able to run in a local Kubernetes environment, such as Minikube on Linux or Docker Desktop on macOS.

Please do not fork the repository or create pull requests. That would give away the solution to other candidates. Please create a new repository and copy the content into your working directory.

**Deliverables:**
- Application changes and new files required to build and run the application in Kubernetes.
- Configured CI pipeline
- Basic observability
- Instructions on how to build, deploy, and test the application locally.

**Bonus:**
- Consider what security improvements are possible and implement respective changes.

**Notes:**
Please do not spend more than 2 hours on this assignment. We are not looking for a perfect production platform, but we are interested in your technical judgment, trade-offs, and ability to communicate clearly.
